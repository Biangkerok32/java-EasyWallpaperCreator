Easy Wallpaper Creator Help


Easy Wallpaper Creator is a utility designed to help quickly create a simple wallpaper image by specifying the generated wallpaper's size, background color, and overlay image.


Optionally, the overlay image may be centered, stretched, or tiled in to the generated image.


Easy Wallpaper Creator is NOT intended to be and WILL NOT be a full blown image editing program. There are too many image editing programs out there to warrant the creation of another one.



Using Easy Wallpaper Creator

When first starting Easy Wallpaper Creator the UI is ready for creating a new wallpaper. Just follow the steps listed below to get started.

  1. Set the desired dimensions ( width and height ) of the wallpaper
      This defaults to the current screen width and height
  2. Choose a background color ( defaults to BLACK )
  3. Choose an overlay image.
  4. Choose how the overlay image is placed on the wallpaper, centered, stretched, or tiled.

At this point there are two things you may do. Select the View->Preview menu item to see a full screen preview of the generated wallpaper or you may select the File->Save menu item to save the generated wallpaper to your hard drive.


Even after saving the generated wallpaper to your hard drive, the View->Preview menu item will still provide a full screen preview of the generated wallpaper.


Should you decide to create another wallpaper, you can select the File->New menu item. This will reset the UI and prepare it for generating a new wallpaper.
